**scanf** - &lt;stdio.h&gt;  
`scanf( <FORMAT STRING>, <VAR 1>, <VAR 2> ... )`  
reads in data from stdin using format string to determine types  

**fgets** - &lt;stdio.h&gt;  
read in from a file stream and store it in a string  
`fgets( <DESTINATION>, <BYTES>, <FILE POINTER> )`
